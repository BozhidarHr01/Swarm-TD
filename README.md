# Swarm TD
A simple 2D Roguelike, Dungeon crawler game with Tower Defence elements
built with Python - pygame-ce

Game Controls:
- Move - WASD
- Attack - Left Mouse Button
- Place objects - 1,2,3,4
- Cancel build mode - E
- Rotate building - R
- Remove building mode - Q
- Zoom in/out - Mouse Scroll Wheel (mostly for testing)
